¿Que es el BoxModel?

El modelo de caja comprende el conjunto de propiedades que definen partes de un elemento que ocupan espacio en una página web, incluyendo el padding y el margin. Básicamente, es un módulo CSS que define cajas rectangulares. Básicamente, es un módulo CSS que define cajas rectangulares. Cada vez que se inserta una etiqueta HTML, se crea una nueva caja rectangular que encierra los contenidos de ese elemento.
